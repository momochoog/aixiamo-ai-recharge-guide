# AIXiamo Gemini / Claude / Grok 充值说明

除了 ChatGPT Plus / Pro，越来越多用户也在关注 Gemini、Claude、Grok、Codex 等 AI 产品。不同产品的账号体系、订阅权益、地区限制、交付方式和售后规则并不完全一样，购买前需要分开判断。

## Gemini 购买前看什么

Gemini 相关服务通常要关注 Google 账号、地区、付款方式、订阅权益和设备使用场景。购买前建议确认：

- 商品是否写清楚适用账号和地区。
- 是否说明开通或交付方式。
- 是否有订单查询和售后入口。
- 是否说明失败后的排查方式。

## Claude 购买前看什么

Claude 用户常关注 Claude Pro、Claude Max、5x、20x、KYC、成品账号等关键词。购买前尤其要看：

- 是订阅权益还是成品账号。
- 是否已过 KYC。
- 是否写清楚登录、改密、绑定和售后边界。
- 100 美元档、200 美元档是否适合自己的使用强度。

## Grok 购买前看什么

Grok 相关产品与 xAI / X 平台生态有关，购买前建议确认：

- 商品对应的账号或订阅类型。
- 使用地区、设备和登录方式。
- 订单交付方式。
- 售后可处理范围。

## Codex / API 购买前看什么

开发者购买 Codex 或 API 相关服务时，应该区分“订阅权益”和“API 额度”：

- 订阅更适合在官方客户端或网页中使用。
- API 更适合集成到 IDE、脚本、Agent、自动化流程。
- API Key 不应放在公开仓库、截图、前端代码或共享文档中。
- 购买前要确认额度、模型、接口地址、稳定性和售后边界。
